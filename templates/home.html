01: Overview
Axiom is a wiki platform for technical and mathematical writing. In its current form, it is intended to be a encylopaedia of decision theoretic findings (axioms, representations, etc). The benefit of Axiom over, for instance, Wikipedia is cleaner integration of TeX like rendering and modern web hyper-referencing.
02: Basic Editing
To edit an article, toggle edit mode by clicking the Editing button on the footer of the page. When edit mode is turned on, click on any paragraph to edit it, or add a new paragraph below. When a paragraph is clicked, the editor box will appear in the bottom of the window. Anything typed in the editor will be immediately (locally) rendered in the corresponding paragraph. To commit changes (save to server) click the Commit button. To revert all changes back to the previous commit, click the Undo button. To delete the paragraph entirely, click the Delete button. To add a new, blank paragraph below the current one, click the Add ↓ button.
Links to other pages are created by delimiting the title of the article (either the title as displayed on the page, or in the url) between double brackets, so that [[completeness]] automatically renders as completeness.
Generic TeX syntax is used wherever possible. To create a new section, open a paragraph and include \section{Section Title}. Section numbers will be generated on the fly, and will update automatically if new sections are added out of order. Hovering over the header title displays a table of contents indexed by sections. The table of contents will be auto updated when sections are added. The first section of each article must be "Overview."
Axiom runs KaTeX for live TeX editing. Generic TeX syntax is used wherever possible. To create a new section, open a paragraph and include \section{Section Title}. To render inline math, delimit TeX markup with $. That is, $f(x) = x^2$ will render as f(x) = x^2.
Display math is delimited by $$: $$\mathbb{E}(f) = \sum_{x \in X}(f)x \pi (x)$$ will render as
\mathbb{E}(f) = \sum_{x \in X} f(x) \pi (x)
(1)
As with section number, equation numbers will be generated and updated automatically.
KaTex will try to catch errors locally. $f(x) = \tau x + (1- \tua) y$ becomes f(x) = \tau x + (1- \tua) y.
03: References
Sections and display equations can be hyper-linked in text, both locally and across pages. To form a hyperlink, the target must have a label. To label an equation or section, add a unique alphanumeric label delimited by single brackets. Section two can be labeled as such: \section{Basic Editing}[basic]; equation one can be labeled as such: $$\mathbb{E}(f) = \sum_{x \in X}(f)x \pi (x)$$[expectation].
Then, to create a hyperlink, use the \ref comand: \ref{sec:basic} will create a link to the cooresponding section: Section 02: Basic Editing. Likewise, \ref{eq:expectation} will render as (1).... Notice that hovering the mouse over relosolved links will display a popup with the rendered equation.
Labeled equations from other articles can also be referenced. For example, the first equation in of the article on the completeness axiom is labeled [cax]. To reference, use the *eq reference, calling first the article name and then the equation label, separated by a semicolon and finally and optional link text, delimited by single brakets: \ref{eq*:completeness;cax[the completeness axiom]}, will render the completeness axiom.... Best practice is to use local links whenever possible to reduce AJAX calls on the server.
Non-local references without the a declared link text will render a default link text when changes are commited, or when [CMD+R] is pressed. The defualt link text is (article.eq#) so the above link becomes (completeness.1)....
04: Popup Overviews
The first section of each article is titled "Overview." Any paragraphs which are included in the article before the next section title are considered the "Overview" of the article. These introductory paragraphs are indended to give a breif description of the content of the article. Particular attention is paid to this becuase, when hovering on interal links (links to other Axiom pages), a popup will display the overview paragraphs.
